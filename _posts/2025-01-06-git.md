【提交】是本地操作，保存你的更改到本地仓库。

【推送】是将本地提交上传到远程仓库。

【共享】通常指的是将本地项目上传到GitHub（包括创建仓库和推送）。

已有仓库推送，需要先配置url

https://github.com/javagitcoder

## 使用 Personal Access Token 认证
生成 Token：

登录 GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic)

点击 Generate new token → Generate new token (classic)

设置权限（至少勾选 repo）

生成并复制 token

使用 Token 推送：

```python
# 方法1：在 URL 中直接使用 token
git remote set-url origin https://javagitcoder:你的token@github.com/javagitcoder/snacks-android.git

# 方法2：推送时使用 token 作为密码
git push -u origin master
# 当提示输入密码时，粘贴你的 token
```
