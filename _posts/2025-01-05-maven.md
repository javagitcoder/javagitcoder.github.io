## pom.xml
```markdown
<dependency>
  <groupId>mysql</groupId>
  <artifactId>mysql-connector-java</artifactId>
  <version>5.1.49</version>
</dependency>
```
```markdown
mvn clean   # 清理编译项目
mvn compile # 编译项目
mvn test    # 测试项目
mvn package # 项目打包
mvn install # 安装到仓库中，会把当前包存放到本地Maven仓库中
mvn clean package -Dmaven.test.skip=true # 跳过测试打包
```
